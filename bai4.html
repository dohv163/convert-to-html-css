ABOUT GRAYSCALE

Grayscale is a free Bootstrap theme created by Start Bootstrap. It can be yours right
now, simply download the template on the preview page . The theme is open source,
and you can use it for any purpose, personal or commercial.

This theme features stock photos by Gratisography along with a custom Google Maps
skin courtesy of Snazzy Maps.

Grayscale includes full HTML, CSS, and custom JavaScript files along with SASS and
LESS files for easy customization!
